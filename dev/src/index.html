デイトラハウスって？

自分らしく人生を歩む一歩に。

デイトラハウスとは、学びと成長を重視する方やキャリアチェンジを考えている方のためのコミュニティ志向のシェアハウスです。
メンバーが仲間と共に自己成長を促進し、新しいスキルを習得し、共に成長していくことを目指しています。

デイトラハウスでは、学びと成長の旅を共に歩む仲間として、皆さんの新たな一歩を応援し、未来への希望を共に育んでいきます。

デイトラハウスでできること

  • 24時間自習室

    24時間利用可能な自習室です。忙しいスケジュールの中でも、自分のペースで学びを追求したい人々に最適な環境を提供します。

  • 無料セミナー

    月に2回無料のセミナーが開催されます。
    参加者は新しい知識やスキルを習得し、業界のトレンドや最新の情報についても理解を深めることができます。

  • デイトラ生との交流

    メンバー同士が交流できるリビングルームやキッチンがあります。定期的なイベントも開催されているので、他のデイトラ生と交流を深めることができます。

Event

  • 01

    対面面談

    デイトラハウスでは、メンバーがより良い生活や学習環境を築くための面談を定期的に開催しています。
    生活環境や施設の利用方法、コミュニティイベントへの参加、学習プログラムへのアクセス、個別のニーズや要望に対するサポート、課題への対処など自由にお話しましょう。

  • 02

    チーム企画

    デイトラハウスのメンバーでチームを作り、案件の獲得を目指します。異なるスキルや専門知識を持つメンバーがチームを構成し、互いの補完し合いながら目標に向かって進んでいきます。

    異なるスキルや専門知識を持つメンバーでチームを結成するため、１人ではできない大きな仕事を受注できることも！

  • 03

    デイトラハウス交流会

    デイトラハウスでは、メンバー同士の親交を深める交流会を開催しています。同じ目標で頑張っている仲間と楽しめるイベントは毎回大好評です！

    実際に行った例では、バーベキューやビンゴゲーム大会などがあります。
